# Description

This repository contains the professional personal landing page for Amanda Williams, created as part of BAIS:3300 — Digital Product Management at the University of Iowa.

The website provides a polished online presence that helps hiring managers and campus recruiters quickly understand Amanda’s academic background, professional experience, and technical skills. The site expands on the traditional resume format by highlighting key internships, analytical capabilities, and strategic thinking.

The landing page emphasizes Amanda’s multidisciplinary perspective and ability to turn data into actionable business insights.

Features

Clean professional landing page layout

Hero section introducing Amanda Williams and her professional focus

Bio section summarizing education and background

Skills section highlighting technical and analytical competencies

Featured projects showcasing internship experiences

Contact section with LinkedIn and professional email

Responsive layout suitable for desktop and mobile viewing

Hosted using Azure Static Web Apps

Getting Started
Prerequisites

To run this project locally you only need:

A web browser (Chrome, Edge, Safari, or Firefox)

Git (optional, if cloning the repository)

Because this is a static HTML/CSS project, no additional dependencies or frameworks are required.

Installation

Clone the repository:

git clone https://github.com/awilliams63/3200-landing-page.git

Navigate into the project folder:

cd 3200-landing-page

Open the site locally by opening the index.html file in your browser.

Example:

open index.html

or double-click index.html.

Usage

This website is designed for recruiters, hiring managers, and professional connections reviewing Amanda Williams as a candidate for roles such as:

Business Analyst

Strategy Analyst

Consulting roles

Sales or revenue strategy roles

Visitors to the site can:

Learn about Amanda’s academic background and interests

Review key skills in analytics and business strategy

Explore internship experiences and professional accomplishments

Access Amanda’s LinkedIn profile or contact her via email
